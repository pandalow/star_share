

## API Overview

This document links to all controller-specific API references. Each module follows the same format used in `Post.md` (description, endpoint, auth, request/response, sample call).

### Available API Docs
- [Post API](Post.md) — drafts, publishing, feeds, metadata, and deletion.
- [Auth API](Auth.md) — verification codes, registration/login, token lifecycle, and `/me` profile.
- [Relation API](Relation.md) — follow/unfollow, relation status, follower/following lists, and counters.
- [OSS API](OSS.md) — pre-signed upload URLs for post content/images.
- [Counter API](Counter.md) — aggregate counts (like/fav) per entity.
- [Action API](Action.md) — authenticated like/fav toggle endpoints.

Add new controller docs in this folder and reference them here to keep the index up to date.
